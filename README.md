# 🛡️ 사장님 든든 (Sajangnim Deundeun) - AI 기반 소상공인 경영 전주기 지원 플랫폼

소상공인의 창업(Plan)부터 운영(Operate), 가치 평가(Value), 그리고 성공적인 마무리(Exit)까지의 전 과정을 AI 데이터 분석을 통해 지원하는 올인원 경영 솔루션입니다.

## ✨ 4대 핵심 서비스 (4 Pillars)

### 1. 🏗️ 스마트 창업 설계 (Plan)
창업 전 사업성 분석과 정교한 계획 수립을 지원합니다.
- **초기 창업 비용 계산기**: 보증금, 인테리어, 권리금 등 초기 투자 비용 산출
- **월간 손익 시뮬레이션**: 매출 목표에 따른 예상 지출 및 순이익 미리보기
- **예상 회수 기간 예측**: 초기 투자금 대비 손익분기점 도달 시점 계산

### 2. 📊 AI 실시간 경영 (Operate)
데이터 기반의 투명한 자금 관리와 전략적 의사결정을 돕습니다.
- **AI 자동 통장 정리**: 입출금 내역을 AI가 사업 수입/지출과 개인 지출로 자동 분류
- **실시간 경영 대시보드**: 매출, 비용, 영업이익 등 핵심 지표 시각화
- **AI 심층 분석 보고서**: 재무 건전성 진단 및 맞춤형 경영 개선 제안
- **지능형 상권 분석**: 주변 경쟁사 분석, SWOT, 4P 마케팅 전략 수립

### 3. 💎 데이터 기반 가치 평가 (Value)
내 사업의 객관적인 현재 가치를 산출하여 자부심과 기회를 제공합니다.
- **기업 가치 산정**: 시설 자산, 재고, 수익 가치를 합산한 종합 평가
- **권리금 배수 적용**: 업종 트렌드와 수익성을 반영한 합리적 권리금 도출
- **가치 평가 보고서**: 매매나 투자 유치 시 활용 가능한 객관적 데이터 제공

### 4. 🚪 전략적 마무리지원 (Exit)
성공적인 마무리를 위한 단계별 가이드와 지원 프로그램을 연결합니다.
- **마무리 체크리스트**: 법률, 세무, 노무 등 폐업/양도 시 필수 이행 사항 관리
- **정부 지원 프로그램**: 희망리턴패키지 등 소상공인 재기 지원금 정보 제공
- **리스크 관리 가이드**: 가산세 방지 및 원상복구 비용 절감 전략

## 🚀 설치 및 실행 방법

### 1. 사전 준비
- **Node.js** 18 이상 설치 ([다운로드](https://nodejs.org/ko))
- **Google Gemini API 키** 발급 ([Google AI Studio](https://aistudio.google.com/apikey))

### 2. 설치 및 실행
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
브라우저에서 `http://localhost:3000`으로 접속합니다.

### 3. 사용 방법
1. **API 키 입력**: 발급받은 Gemini API 키를 입력합니다 (브라우저 로컬 저장).
2. **사업 정보 설정**: 상호명, 대표자명, 업종을 선택합니다.
3. **데이터 업로드**: 은행 입출금 내역(Excel/CSV)을 업로드하여 분석을 시작합니다.
4. **4대 서비스 활용**: 상단 탭을 통해 창업 설계부터 가치 평가까지 자유롭게 이동하며 활용합니다.

## 🔧 기술 스택
- **Frontend**: React 19, TypeScript, Vite
- **AI Engine**: Google Gemini AI (Google Generative AI SDK)
- **Styling**: Tailwind CSS, Framer Motion (애니메이션)
- **Visualization**: Recharts (데이터 시각화), Lucide React (아이콘)
